## Installation & Setup

1. Clone the repository.
2. Install dependencies:

   ```bash
   npm install
   ```

3. Create a `.env` file and add your environment variables:

   ```bash
   PORT=3000
   ```

4. Start the server:

   ```bash
   node index.js
   ```
